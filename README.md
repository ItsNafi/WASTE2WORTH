# WASTE2WORTH

WASTE2WORTH is a circular economy web application for scrap recycling, upcycled crafts, and community campaigns.

## Features

- User authentication and role-based dashboards for Admin, Citizen, BhangariShop, Creator, and Volunteer
- Scrap listing and buying flows
- Pollution reporting and admin monitoring
- Craft creation and storefront browsing
- Campaign management and reward certificates

## Setup

1. Install dependencies:

```bash
npm install
```

2. Create a `.env` file from `.env.example` and set your database credentials and JWT secret.

3. Start the application:

```bash
npm start
```

4. Open the app in your browser at `http://localhost:3000`.

## Development

- `npm start` - start the server
- `npm run dev` - start the server with automatic restart on changes
